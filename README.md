# 프로젝트 협업 앱
> 2021년도 수원대학교 정보통신학부 졸업작품입니다.
  

## 프로젝트 내용
> 사용자들간의 협업을 할수 있도록 할 수 있는 앱입니다.

- 구성원 간의 일정 공유, 채팅, 파일 공유
- 팀원 검색, 프로젝트 검색
- 

  


### 멤버
- 박수빈 
- 박종건
- 나선재

### 기술스택
- Flutter
- Spring Boot
- MySql
